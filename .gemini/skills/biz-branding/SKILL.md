# ✨ AI Product Branding Agent (인스타 브랜딩)

경로: `/style-transfer`
API: `src/app/api/style-transfer/route.ts`
사용 AI: Gemini 2.5 Flash (텍스트) + FAL AI nano-banana-2 (이미지 생성)

## 에이전트 체인
1. 광고 전문가 (Ad Creative Director)
   - 제품명 + 스타일 힌트 기반 브랜드 포지셔닝 분석
   - FAL AI용 상세 영어 프롬프트 생성
   - 인스타 업로드 가이드 생성
2. FAL AI 이미지 생성기
   - 제품 이미지 + 스타일 프롬프트로 브랜딩 이미지 변환

## 핵심 설계 원칙
- Gemini Vision 사용하지 않음 (429/차단 이슈 회피)
- 텍스트 기반 프롬프트만으로 스타일 결정
- 레퍼런스 이미지 업로드 없음 — 에이전트가 자체 판단

## 📸 이미지 스타일 가이드 (mobileeditingclub 참고)

프롬프트에 내장된 스타일 방향:
- 모델이 제품을 자연스럽게 들고 있는 라이프스타일 샷
- 소프트 내추럴 라이팅, 약간 따뜻한 톤 (warm beige, soft cream)
- 미니멀하고 깔끔한 배경 (단색 또는 실내 인테리어)
- 제품이 화면의 주인공이 되는 클로즈업~미디엄샷
- 실제 촬영처럼 보이는 리얼리스틱 퀄리티
- 비율: 4:5 (1080x1350px) — 피드에서 가장 큰 면적 차지

스타일 프리셋:
- 미니멀 스튜디오 / 라이프스타일 / 럭셔리 에디토리얼
- 내추럴 아웃도어 / 네온 사이버펑크 / AI 자동 추천

## 📱 인스타그램 마케팅 전략 (2026 알고리즘 기반)

이 섹션은 프롬프트 작성 및 마케팅 가이드 생성 시 반드시 참조해야 하는 최신 인스타그램 전략입니다.
출처: [Aurelius Media 2026 Instagram Guide](https://www.aureliusmedia.co/blog/what-works-on-instagram-2026) 기반으로 재구성.

### 알고리즘 핵심 (2026)
- 인스타그램은 단일 알고리즘이 아님 — Feed, Reels, Stories, Explore 각각 별도 AI 랭킹 시스템
- 랭킹 시그널 우선순위: 시청 시간(Watch Time) > DM 공유(Sends per Reach) > 좋아요(Likes)
- 모든 콘텐츠의 리트머스 테스트: "누군가 이걸 친구에게 DM으로 보낼까?"

### 콘텐츠 포맷별 전략
| 포맷 | 용도 | 성과 |
|------|------|------|
| Reels | 신규 도달 & 발견 | 2.35배 더 많은 도달, 인앱 시간의 50% |
| 캐러셀 | 참여도 & 저장 | 평균 참여율 10.15%, 최대 20장, 미스와이프 시 재노출 |
| 정적 사진 | 브랜드 아이덴티티 | 제작 속도 빠름, 도달 천장 낮음 |
| 스토리 | 관계 강화 & 충성도 | 팔로워 전용, 인터랙티브 스티커 활용 |

### 인스타 SEO (해시태그 → 키워드)
- 해시태그 최대 5개로 제한됨, 3~5개 니치 해시태그만 사용
- 키워드 최적화 캡션이 해시태그 대비 약 30% 더 많은 도달
- SEO 스택: 유저네임, 바이오, 캡션, 대체텍스트, 릴스 음성, 화면 텍스트 모두 키워드 포함
- 공개 프로페셔널 계정 게시물은 구글 검색에도 노출됨

### 최적 게시 시간
- 화요일 오후 1~7시, 수요일 낮 12시~밤 9시가 최고
- 릴스는 배포 수명이 길어 시간 덜 민감
- 캐러셀은 출근 시간(7~9AM), 점심(11AM~1PM)이 효과적
- 토요일이 참여도 가장 낮음

### 콘텐츠 제작 원칙
1. DM 공유를 위한 디자인 — 공감, 유용한 팁, 토론거리, "너 생각나서" 콘텐츠
2. 처음 3초 후킹 — 움직임, 굵은 텍스트, 패턴 인터럽트
3. 캡션은 검색용으로 작성 — 대화체, 키워드 풍부, 미니 랜딩페이지처럼
4. 캐러셀은 사진+짧은 영상 혼합 (하이브리드 캐러셀)
5. 니치에 집중 — 알고리즘이 "토픽 클러스터"를 구축, 일관성이 핵심
6. 오리지널 콘텐츠만 — 30일 내 10회 이상 리포스트 시 추천에서 제외
7. Trial Reels 활용 — 비팔로워에게 먼저 테스트 후 성과 좋으면 전체 배포
8. 긴 캡션이 짧은 캡션보다 성과 좋음 (캡션 체류 시간 = 가치 시그널)

### 제품 브랜딩 이미지 → 인스타 업로드 가이드 생성 시 포함할 내용
- 추천 게시물 타입 (피드/릴스/캐러셀/스토리 중 제품에 맞는 것)
- 최적 업로드 시간대 (제품 카테고리 + 타겟 기반)
- 캡션 구조 가이드 (후킹 첫 줄 → 핵심 가치 → CTA)
- 키워드 SEO 전략 (캡션에 넣을 검색 키워드 제안)
- 타겟 오디언스 설정 팁
- DM 공유를 유도하는 콘텐츠 전략

## 수익화 방향
- 제품 브랜딩 이미지 대량 생성 → 광고 소재 제작 비용 절감
- 인스타 피드 일관성 유지를 위한 스타일 템플릿 시스템
- 마케팅 에이전시 대상 B2B 화이트라벨

## 프롬프트 수정 시
`src/app/api/style-transfer/route.ts`의 `adDirectorPrompt` 변수 수정.
스타일 가이드 섹션을 변경하면 생성 이미지 톤이 바뀜.
마케팅 가이드 품질을 높이려면 위 인스타 전략 섹션의 내용을 프롬프트에 반영.

## 향후 개선
- 카드뉴스 에이전트와 연동: 브랜딩 이미지 위에 타이포그래피 오버레이 (CSS 렌더링)
- 브랜드 컬러/폰트 설정 저장 기능
- 시리즈 생성: 같은 제품으로 여러 스타일 변형 한번에 생성
- 캐러셀 자동 생성: 제품 이미지 여러 장 → 하이브리드 캐러셀 구성
