# 🧪 QA 테스터 (Quality Assurance)

각 에이전트 기능을 테스트할 때 아래 체크리스트를 따르세요.

## AI Shorts Agent (`/`)

- [ ] 트렌드 데이터가 로드되는가?
- [ ] 키워드 클릭 시 주제가 선택되는가?
- [ ] "주제 추천" 버튼 → 5개 주제가 생성되는가?
- [ ] 주제 선택 후 "파이프라인 시작" → 4단계 순차 실행되는가?
- [ ] 각 단계 결과가 화면에 표시되는가?
- [ ] "노션에 저장" → 성공 토스트가 뜨는가?
- [ ] 히스토리 저장/불러오기가 동작하는가?

## Card News Agent (`/card-news`)

- [ ] "주제 추천" → 3개 주제 카드가 표시되는가?
- [ ] 주제 클릭 → 4단계 파이프라인이 실행되는가?
- [ ] CardPreview 컴포넌트가 정상 렌더링되는가?
- [ ] 카드 텍스트 인라인 편집이 실시간 반영되는가?
- [ ] 이미지 다운로드(JPG)가 동작하는가?
- [ ] 노션 저장이 동작하는가?
- [ ] 노션 히스토리에서 "불러오기"가 동작하는가?

## Product Branding Agent (`/style-transfer`)

- [ ] 제품 이미지 업로드가 동작하는가?
- [ ] 제품 이미지 삭제(✕) 버튼이 동작하는가?
- [ ] 스타일 프리셋 선택이 동작하는가?
- [ ] 직접 스타일 입력이 동작하는가?
- [ ] "AI 브랜딩 시작" → 로딩 오버레이 → 결과 표시되는가?
- [ ] 결과 이미지 다운로드가 동작하는가?
- [ ] 인스타 업로드 가이드 아코디언이 동작하는가?
- [ ] 노션 저장이 동작하는가? (413 에러 없이)
- [ ] 히스토리 저장/불러오기가 동작하는가?

## 공통

- [ ] 429 에러 시 API 키 로테이션이 동작하는가?
- [ ] 네트워크 에러 시 사용자에게 에러 메시지가 표시되는가?
- [ ] 모바일 반응형이 깨지지 않는가?
- [ ] 노션 DB 속성(channel, 상태)이 올바르게 설정되는가?

## 에러 시나리오

- [ ] Gemini API 키가 없을 때 → 명확한 에러 메시지
- [ ] FAL AI 키가 없을 때 → 명확한 에러 메시지
- [ ] 노션 키가 없을 때 → 명확한 에러 메시지
- [ ] 이미지 용량 초과 시 → 413 대신 사용자 친화적 메시지
